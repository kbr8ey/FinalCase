✓ Containerization (Docker & DevOps)

  - Structured logging with severity levels

  - Audit trail for cart operations

✓ Data Modeling
  - Product dataclass with type hints
  - JSON serialization/deserialization
  - In-memory cart management

DATA FLOW
=========

1. User loads home.html in browser
2. JavaScript calls GET /products from Flask API
3. Flask responds with JSON array of 8 products
4. Frontend renders responsive product grid
5. User clicks "Add to Cart"
6. JavaScript sends POST /cart/add with user_id, product_id, quantity
7. Flask stores in-memory cart and responds with confirmation
8. Frontend updates cart counter and shows success notification
9. User can view cart contents and proceed to checkout
